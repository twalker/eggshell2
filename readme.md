eggshell2
=========

A sandbox for learning, assembling, and making sense of select microlibraries

- "framework": Flux or Ampersand (models, router, etc.)
- modules: browserify or es6 modules, or system.js if needed.
- Modles: stores??
- Views: react
- task runner: npm
